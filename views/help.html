Project Management

Create a new project

1) To add a new project, go to the option "Projects" in the side menu and then press the "add new" button.

2) Then proceed to complete the form fields and press the "save" button.

3) Now you can see the new project added on the main page. The projects that other users have shared with you also appear here, however you will not be able to share them because you are not the owner.

Edit a project

1) To edit a project press the button with the pencil icon in the corresponding row of the table.

2) Then proceed to edit the form fields and press the "save" button.

Delete a project

1) If you want to delete a project simply press the trash icon in the corresponding column of the table. Once pressed, the project will disappear from the table along with its associated tasks.

Share a project

1) To share a project press the share icon.

2) In the new screen press the button "add new user".

3) In the pop-up window choose the user to whom you want to share the project and press the button "add user".

4) Then you can see the new user added in the table. If you want to delete the user just press the trash icon.




Task Management

Create a new task

1) To add a new task, go to the option "Tasks" in the side menu and then press the "add new" button.

2) Then proceed to complete the form fields and associate the task with the correct project using the selection field, press the "save" button.

3) Now you can see the new task added on the main page.

Edit a task

1) To edit a task press the button with the pencil icon in the corresponding row of the table.

2) Then proceed to edit the form fields and press the "save" button.

Delete a task

1) If you want to delete a project simply press the trash icon in the corresponding column of the table. Once pressed, the task will disappear from the table.

Task timer

1) To start counting the time invested in a task, press the "start timer" button, the "start date" will appear in the table and the "status" of the task will change.

2) To stop the task timer, press the "stop timer" button in the row of the corresponding task. Once pressed, the end date and time will be added, the status will change to finished and the time spent will be displayed.

3) In the project management screen we can see how the task that we have just finished appears in the "finished tasks" column, and the progress is 100% since the project only has that task associated with it.
